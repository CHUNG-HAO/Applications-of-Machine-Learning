## 前言
由於現今全台近視率越來越高，是一個趨勢，目前台灣每５人就有4人近視，為了解決問題，我們將近視普及的原因設置在坐姿方面，科技業的興起，也造就不少學生、上班族對於坐在電腦桌前面的時間越藍越久，成為近視的推手之一，為了解決這個問題，將訓練一個模型針對“影線辨識坐姿健康管理系統”去改善這個問題。
